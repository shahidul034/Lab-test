Microprocessor Lab Test - 2023

    A section

  • Write an assembly program to take two strings and concatenate the first two characters of the first string and the last two characters of the second string.

    Input: abc dma

    Output: abma

    Input: kdb mark

    Output: kdrk

  • Find the square number in the sequence.

    Input: 2 7 9 16

    Output: 9 16

  • Develop an assembly program to encrypt and decrypt a string using a simple algorithm like the Caesar cipher. The program should be able to perform decryption.

    Input: def 3

    Output: abc

B section

  • Find the common factors of two numbers.

    Input: 48 54

    Factors of 48: 1, 2, 3, 4, 6, 8, 12, 16, 24, 48

    Factors of 54: 1, 2, 3, 6, 9, 18, 27, 54

    common factors: 1,2,3,6

  • Delete the character using the particular index, and print the string after removing the character.

    Input: abcd 1

    Output: acd

    Input: abcd 0

    Output: bcd

  • Develop an assembly program to encrypt and decrypt a string using a simple algorithm like the Caesar cipher. The program should be able to perform encryption.

    Input: abc 3

    Output: def
